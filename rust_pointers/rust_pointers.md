# Rust Pointers

## Toy Model: Pointer Pattern Exercises
Small, focused exercises to build muscle memory for raw pointer operations in Rust.

## Goal
Learn to predict which combination of `&`, `*`, `const`, `mut`, `as_ptr`, casts to use.

## Adding Exercises
Ask for more exercises anytime. They'll be appended to `src/lib.rs`. Can request:
- More practice on tricky patterns
- Variations (mutable vs const, different types)
- Advanced patterns (pointer arithmetic, `offset()`, slices from raw parts)
