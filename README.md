# toymodels

In this repo [mchels](https://github.com/mchels) attempts to learn anything and everything
by building toy models with LLMs as an instructor. Imagine following a high-quality tutorial
like [Writing an OS in Rust](https://os.phil-opp.com/), but for any topic you want and with
tailored hints and guidance.

See CLAUDE.md and my blog post [LLMs as instructors](https://morten.hels.dk/llms-as-instructors.html) for more.
